# app-ui

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### To run docker file
```
docker build -t app-ui:local-1 . 
```

### To run docker file
```
docker-compose up -d 
```
